# SuperAdmin Boot

La pantalla `public_html/superadmin/index.php` muestra:

- estado general;
- última ejecución;
- hostname, kernel y LAN;
- load, RAM, disco, updates, reboot requerido;
- servicios fallidos;
- estado de Telegram;
- historial reciente;
- contratos y paths.

Restricción: no ejecuta comandos del sistema. Solo lee JSON persistido.
